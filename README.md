# AI_Crop_Disease_Prediction_System
An AI-powered application that identifies crop diseases from leaf images using a CNN model and provides actionable organic and chemical management strategies. Developed as a B.Tech Minor Project (BTCS607N).
# 🌱 AI-Driven Crop Disease Prediction and Management System

![Project Status](https://img.shields.io/badge/Status-In%20Development-yellow)

## 📌 Project Overview
Farmers often face significant crop yield losses due to the late or inaccurate identification of plant diseases. Manual inspection is time-consuming, requires specialized agricultural knowledge, and is prone to human error. 

This project is an AI-driven application that allows farmers and agricultural experts to upload images of plant leaves. The system utilizes a Convolutional Neural Network (CNN) to accurately predict the disease and provides immediate, actionable management strategies (such as pesticide or organic fertilizer recommendations) to mitigate crop damage.

## ✨ Key Features
* **📸 Image Acquisition:** Capture real-time images via the device camera or upload existing photos from the gallery.
* **🧠 AI Prediction Engine:** Fast and accurate image classification utilizing a trained deep learning model, complete with confidence scores.
* **💊 Treatment & Management Guides:** Provides tailored, actionable solutions—including both organic and chemical treatments—based on the specific disease identified.
* **📂 Scan History:** Automatically logs past scans and predictions for easy tracking and reference by the user.
* **🛡️ Admin Dashboard:** Allows administrators to manage users, update the disease database, and flag low-confidence images for future model retraining.
